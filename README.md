# Pocket Quest (Java Swing)

A medium-scope, Pok√©mon-like 2D game demo made in pure Java.

## Features
- Tile-based overworld with camera follow
- NPC interactions with dialogue (`E` key)
- Random wild encounters in tall grass
- Simple turn-based battle system
- Leveling, HP, XP, and rival trainer battle

## Run
```bash
javac PokemonLikeGame.java
java PokemonLikeGame
```

## Controls
- Move: `W A S D` or Arrow keys
- Interact: `E`
- Battle actions: `1` Attack, `2` Strong Attack, `3` Heal, `4` Run
